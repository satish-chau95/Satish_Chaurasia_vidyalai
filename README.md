## Getting Started

Step 1: Install packages and make sure you are using Node.js version 18.20 or higher.

```bash
npm i
```

Step 2: Run the development server:

```bash
npm run dev
```

Step 3:
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result .
